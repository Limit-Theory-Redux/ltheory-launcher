# LTR Launcher
A simple Limit Theory Redux launcher written with use of the Tauri toolkit (Rust, Nuxt.js/Vue.js) currently targeted at Windows only.

# Development
## Prerequisites:
- [Rust](https://www.rust-lang.org/)
- [Tauri](https://tauri.app/)
- [pnpm](https://pnpm.io/)

## Developing the app
`pnpm install`
`pnpm tauri dev`

## Building the app
`pnpm tauri build`
